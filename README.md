# QuizMaster

A mobile quiz application built with Swift using MVVM architecture.

## Architecture

The project follows the MVVM (Model-View-ViewModel) architecture pattern:

- **Models**: Data models and business logic
- **Views**: UI components and screens
- **ViewModels**: View state management and business logic
- **Services**: API services, data persistence, and other services
- **Utils**: Helper functions and extensions
- **Resources**: Assets, fonts, and other resources

## Requirements

- iOS 15.0+
- Xcode 13.0+
- Swift 5.5+

## Installation

1. Clone the repository
2. Open `QuizMaster.xcodeproj` in Xcode
3. Build and run the project
